# Template
The template repository, used to speed up the process of creating additional repositories.

## Rails reference

### Setup guide here
https://guides.rubyonrails.org/install_ruby_on_rails.html#install-ruby-on-ubuntu

*Alternative site (might be better)*
https://documentation.ubuntu.com/server/how-to/web-services/install-ruby-on-rails/

### Running development server

Once you have the repo cloned, `cd NIPS` to move into the Rails root directory, and run `rails s`
to begin a development server from your local device. 

--
## WSL reference

WSL is a lightweight virtual machine. It lives on your local machine, and is a *seperate operating system*. Once installed, open the WSL terminal and clone the repository inside WSL. Begin setup from here.
