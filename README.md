# Template

### Setup Scripts

The `/setup_scripts` directory contains a bash (linux/macOS) script

`setup.sh`

And a powershell (windows) script

`setup.ps1`

Run the appropriate script for your operating system when you first clone this repository, and it should never need to be run again. These scripts merely setup some automation on your local clone.
