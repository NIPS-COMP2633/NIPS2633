# Template
The template repository, used to speed up the process of creating additional repositories.

## Rails setup 

### Setup guide here
https://guides.rubyonrails.org/install_ruby_on_rails.html#install-ruby-on-ubuntu

### Running development server
Once you have the repo cloned, `cd NIPS` to move into the Rails root directory, and run `rails s`
to begin a development server from your local device.
